## LoyalFlow Code Review Skill

### Key Focus Areas
- TypeScript strict mode compliance
- React Server vs Client component usage
- React Hook Form implementations
- Prisma query efficiency and safety
- API route authentication/authorization
- Security issues (JWT exposure, SQL injection)
- Error handling patterns
- TanStack Query hook usage

### Typical Triggers

Codex should use this skill when:
- Reviewing pull requests
- Implementing new React components
- Writing Prisma queries
- Building API routes
- Fixing security vulnerabilities
- Optimizing data fetching with TanStack Query
