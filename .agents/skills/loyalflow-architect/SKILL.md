## LoyalFlow Architect Skill

### Responsibilities
- Validate project structure against Next.js 16 App Router best practices
- Ensure multi-tenancy isolation through Prisma models and route guards
- Review authentication flow (NextAuth v5 + JWT)
- Monitor database efficiency (index usage, query optimization)
- Advise on scalable API patterns
- Guide Prisma schema evolution
- Verify separation of concerns (components, lib, routes)
- Architect data flow between TanStack Query, React Hook Form, and backend

### Triggers

Triggers automatically when:
- Discussing system architecture changes
- Evolving Prisma schema
- Reviewing API route design
- Planning scalability improvements
- Validating multi-tenancy implementation
