## LoyalFlow Security Engineer Skill

### Key Responsibilities
- Review and enforce security best practices
- Validate multi-tenancy data isolation
- Implement OWASP Top 10 protections
- Review authentication and authorization flows
- Ensure secure API endpoints
- Audit data validation and sanitization
- Implement rate limiting and DDoS protections
- Verify secure cookie and session handling

### Trigger Conditions

Automatically activates when:
- Discussing security architecture
- Reviewing authentication flows
- Implementing API protections
- Validating multi-tenancy isolation
- Addressing OWASP vulnerabilities
- Configuring rate limiting
- Reviewing session handling
