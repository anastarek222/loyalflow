## LoyalFlow QA Testing Skill

### Core Responsibilities
- Maintain Playwright end-to-end test suite
- Create tests for core flows: authentication, business onboarding, customer lifecycle, loyalty earning, reward redemption, dashboard analytics, and settings
- Implement visual regression testing
- Monitor test reliability and flakiness
- Validate user flows end-to-end
- Ensure test coverage for critical paths

### Trigger Conditions

Automatically activates when:
- Creating or updating Playwright tests
- Investigating flaky tests
- Validating feature implementation
- Running test suites
- Discussing test coverage
